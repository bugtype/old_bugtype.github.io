## 기본지식

### 네트워크 포트

21 - ftp

22 - SSH (Secure shell)

23 - telnet

25 - SMTP mail

80 - http

443 - SSL( Secure Socket Layer )


### HTTP 통신

- METHOD -GET, POST, HEAD 등
- PROTOCOL - TCP, UDP
- SSL
